Quark supports multiple platforms. You can find all of them in the table below.

| Platform Name                             | Compatible with                  | Compatible version        |
|-------------------------------------------|----------------------------------|---------------------------|
| `bukkit`                                  | Bukkit / Spigot / Paper / Purpur | `1.8.8` - `1.21.8`        |
| `paper`                                   | Paper / Purpur / other forks     | `1.19.3` - `1.21.8`       |
| `velocity`                                | Velocity                         | `3.4.0`                   |
| `bungeecord`                              | BungeeCord / Waterfall           | `1.19-R0.1` - `1.21-R0.4` |

<Callout title="How to use it?" type="info">
  To use Quark with a specific platform, you need to add the platform dependency to your project. See the [Dependencies](/docs/quark/getting-started/dependencies) page for more information.
</Callout>
