title: What is Quark?
icon: Presentation
---

Quark is a lightweight, runtime dependency management system for plugins running on Minecraft server platforms. 
Providing a simple API, Quark makes it easy to download dependencies during your plugin startup without requiring shading or complex setup procedures.

## Cross-Platform Integration

Quark seamlessly integrates with various platforms, making it effortless to manage dependencies across different systems.
Whether you're developing for Bukkit, Paper, BungeeCord, Sponge or Velocity, Quark has you covered.

See the [Platforms](/docs/quark/supported-platforms) page for more information.

## Gradle Plugin

Quark provides a Gradle plugin that simplifies the integration of Quark into your project, allowing you to easily manage dependencies and configurations, without requiring complex setup procedures.
